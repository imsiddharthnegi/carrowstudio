# Carrow Studio

A premium marketing agency website showcasing strategy, design, and results-driven marketing. Built as a portfolio demo to demonstrate modern web design and development practices.

## Overview

Carrow Studio is a fictional marketing agency website designed to showcase professional branding, intentional design, and grounded copywriting. The site emphasizes selectivity, strategic thinking, and outcome-focused marketing.

## Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Package Manager:** npm/yarn

## Features

- **Responsive Design:** Mobile-first approach with seamless desktop experience
- **Modern Aesthetics:** Glassmorphism effects, minimal design language, premium feel
- **Optimized Performance:** Built with Vite for fast development and production builds
- **Clean Architecture:** Organized component structure for maintainability

## Sections

- **Hero:** Grounded headline with clear value proposition and CTA
- **About Studio:** Authentic positioning with curated workspace imagery
- **Studio Projects:** Conceptual work showcasing design thinking and approach
- **CTA Section:** Results-focused call-to-action with intentional messaging
- **Footer:** Transparent attribution with professional navigation and links

## Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/carrow-studio.git

# Navigate to project directory
cd carrow-studio

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

The site will be available at `http://localhost:5173/`

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
```
